How to use the Playwright Recorder

Step 1: Navigate to the target webpage

Use the address bar to go to the taget website.

Step 2: Interact with the page

As you interact with the elements in the page the recorder will generate code in the 2nd window opened.

Step 3: Copy the code

Copy the code from the 2nd window opened, paste it in your target file and tweak it as needed.

-- close the browser window when finished.

Did you know?

It's also possible to start the Playwright recorder directly in your code by using the pause() method in the Playwright page object and manually clicking the Record button in the 2nd window opened.

Example using the robocorp-browser library: 

from robocorp import browser

browser.configure(headless=False)
browser.goto("http://google.com")
browser.page().pause()
